Add CSS code in the Back-end and Front-end easily

== Description ==
Make Css code in your administration, as you can also add additional CSS code in your website in a very easy way.

<h4>This are some of the options Plugin: </h4>
<ul>
<li>Add css code for the Front-end</li>
<li>Add css code for the Back-end (Update 2 times)</li>

</ul>

Gives shape and life to your WP-Admin and Page with 'Adding CSS in Wordpress' ;)

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Adding CSS in Wordpress'
3. Activate 'Adding CSS in Wordpress' from your Plugins page.
4. Visit 'Setting > 'Adding CSS'.
